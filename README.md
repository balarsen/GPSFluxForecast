# GPSFluxForecast
Work on the forecasting of GPS fluxes

## Predict GPS fluxes using K-D Trees

Details:
* 8 hour CXD Fluxes form Morley
* 1 energy: 2 MeV
* 1 L: L = 4-4.5
* 2001-2016 (or more)
* log fluxes (standardized?)
* choose 10 intervals
    * 5 storms
    * 5 non storms
* interval, 7 days prior, 21 days forward
